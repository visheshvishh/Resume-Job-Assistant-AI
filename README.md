# Resume-Job-Assistant-AI
ResumeAI is a web-based ATS resume analyzer that compares resumes with job descriptions to generate a match score, identify skill gaps, and provide actionable feedback. It supports PDF/DOCX uploads, offers smart keyword analysis, and runs fully client-side for fast, secure, and private use.

# 🚀 ResumeAI — ATS Resume Analyzer

A modern **AI-powered Resume Analyzer** that compares your resume with a job description and generates an **ATS (Applicant Tracking System) score**, detailed feedback, and job opportunities.

---

## 📌 Overview

ResumeAI helps job seekers **optimize their resumes** by analyzing:

* Skill match with job description
* Resume structure & formatting
* Experience relevance
* Education alignment

It provides **actionable insights** to improve your chances of getting shortlisted.

---

## ✨ Features

### 📄 Resume Upload & Parsing

* Supports **PDF, DOCX, TXT**
* Drag & Drop file upload
* Extracts and previews resume content
* Uses **PDF.js** for PDF parsing 

---

### 🧠 Smart ATS Analysis

* Matches **50+ tech keywords**

* Detects:

  * Skills
  * Experience
  * Education
  * Resume format

* Generates **ATS Score (0–100)**

---

### 📊 Scoring System

| Criteria       | Weight |
| -------------- | ------ |
| Keyword Match  | 45%    |
| Experience     | 25%    |
| Format Quality | 15%    |
| Education      | 15%    |

Defined using centralized configuration 

---

### 📈 Results Dashboard

* Circular score visualization
* Matched vs Missing skills
* Resume insights (name, email, experience)
* Feedback & improvement suggestions

---

### 💡 Smart Feedback System

* ❌ Missing skills alerts
* ⚠️ Optimization suggestions
* ✅ Strength highlights
* ℹ️ ATS tips

---

### 💼 Job Search Integration

* Generates job links from:

  * LinkedIn
  * Naukri
  * Indeed
  * Google Jobs

---

### 📤 Export Report

* Download full analysis as `.txt`
* Includes:

  * Score breakdown
  * Feedback
  * Skills analysis

---

### 🌙 UI/UX Features

* Dark/Light theme toggle
* Responsive design
* Smooth animations
* Clean modern UI using CSS variables

## ⚙️ How It Works

1. User uploads resume or pastes text
2. Job description is provided
3. System extracts:

   * Keywords
   * Experience
   * Education
4. Compares resume vs job description
5. Calculates weighted score
6. Displays results + feedback

Detailed workflow and architecture explained in project docs
